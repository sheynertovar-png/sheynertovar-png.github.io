⚖️ Balance de Entropía

Esta sección permite análisis avanzado del balance de entropía según Cengel & Boles.

Ecuación del Balance:
ΔS = Q/T + Sgen

Donde:
• ΔS = Cambio de entropía del sistema
• Q = Calor transferido
• T = Temperatura
• Sgen = Generación de entropía

📚 Teoría Fundamental

Ecuaciones Tds (Gibbs)

T·ds = du + P·dv

T·ds = dh - v·dP

Referencia: Cengel & Boles, Capítulo 7

Segunda Ley de la Termodinámica

ΔS_universo ≥ 0

Para un proceso reversible: ΔS_universo = 0

Para irreversible: ΔS_universo > 0

Generación de Entropía

Sgen = ΔS - Q/T

Si Sgen = 0 → Reversible

Si Sgen > 0 → Irreversible (Posible)

Si Sgen < 0 → Imposible

❓ Ayuda y Referencia

¿Cuál es la diferencia entre versión estándar y PRO?

La versión PRO incluye: Balance de entropía, Generación de entropía, 8 gases diferentes, Diagrama P-v, Exportación a PDF, y análisis de reversibilidad automático.

¿Cómo interpreto Sgen?

Sgen = 0: Proceso reversible

Sgen > 0: Proceso irreversible pero posible

Sgen < 0: Proceso imposible (violaría la Segunda Ley)
